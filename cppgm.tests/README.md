# cppgm.tests

Add assignment-local course tests under `course/paN`, where `N` is the
assignment number. Each `paN` directory has a `course` symlink back to
`cppgm.tests/course`, so tests added here are visible from that assignment as
`course/paN`.

Most assignments use a flat `course/paN/*.t` layout matching their local
`tests/` directory. Assignments with multiple harness buckets use matching
subdirectories:

- `course/pa33/preproc` and `course/pa33/compile`
- `course/pa34/link`
- `course/pa35/o0`, `course/pa35/o1`, `course/pa35/o2`,
  `course/pa35/driver/o1`, and `course/pa35/driver/o2`
- `course/pa36/o1` and `course/pa36/o2`

The optional debug-info suites mirror those layouts under
`course/pa35/debuginfo` and `course/pa36/debuginfo`.

Reference output files generated by the harness use `.my*` names and are
ignored locally.
